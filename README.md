## Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/xkevan19/plants.git
   ```
2. Navigate to the project folder:
   ```sh
   cd plantos
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Start the development server:
   ```sh
   npm run dev
   ```
